# typescriptBasicUdemy

## Settings
- install typescript  
`$ npm install --save-dev  typescript@4.7.4`

## npm command
- Create package.json  
`$ npm init -y`

## Git command
- Create branch  
`$ git checkout -b [branch name]`
- Push  
`git push -u origin HEAD`
- checkout(前のブランチ)  
`$ git checkout -`
- merge(前のブランチ)  
`$ git merge -`

- Pull  
`$ git pull origin master`
- Fetch  
`$ git fetch origin`
