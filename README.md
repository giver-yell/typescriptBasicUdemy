## npm command
- Create package.json  
`$ npm init -y`

## Git command
- Create branch  
`$ git checkout -b [branch name]`
- Pull  
`$ git pull origin master`
- Push  
`git push -u origin HEAD`
- checkout(前のブランチ)  
`$ git checkout -`
- merge(前のブランチ)  
`$ git merge -`
